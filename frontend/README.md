# 出勤加班單系統 Frontend

本專案為一個 React + TypeScript 應用程式，旨在幫助使用者上傳 CSV 格式的出勤記錄，自動計算加班時數與誤餐費，並提供篩選、下載 Excel/PDF 報告及列印的功能。

## 專案設定

1.  進入 `frontend/` 目錄：
    ```bash
    cd frontend
    ```
2.  安裝依賴套件：
    ```bash
    npm install
    ```

## 啟動開發伺服器

```bash
npm run dev
```
應用程式將在瀏覽器中啟動，通常位於 `http://localhost:5173`。

## 執行測試

```bash
npm test
```

## 功能說明

1.  **檔案上傳**: 點擊「上傳 CSV 檔案」按鈕，選擇符合格式的 CSV 檔案。
    *   **CSV 格式要求**: 檔案必須包含標頭行，且欄位順序必須為：`員工編號,姓名,歸屬日期,上班時間,下班時間`。
    *   系統會自動跳過空白行或格式不正確的行。
2.  **加班與誤餐費計算**:
    *   **平日 (週一至週五)**: 加班從 18:00 開始計算。若下班時間晚於或等於 19:30，則提供 50 元誤餐費。
    *   **假日 (週六、週日)**: 當日所有工時皆視為加班，不提供誤餐費。
3.  **篩選**:
    *   **按姓名篩選**: 在輸入框中輸入員工姓名，結果將即時篩選。
    *   **按日期篩選**: 選擇開始日期和結束日期，將顯示該日期範圍內的記錄。
4.  **報告下載與列印**:
    *   **下載 Excel**: 將目前顯示的加班記錄匯出為 Excel (.xlsx) 檔案。
    *   **下載 PDF**: 將目前顯示的加班記錄匯出為 PDF (.pdf) 檔案。
    *   **列印報告**: 開啟瀏覽器列印功能，列印目前顯示的報告。
